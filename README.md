1. Clone the temp-TLDA_Model git repository
```bash
git clone https://github.com/wan-mlab/temp-TLDA_Model.git
```
2. Navigate to the directory of temp-TLDA_Model package
```bash
cd /your path/temp-TLDA_Model
pip install .
```
